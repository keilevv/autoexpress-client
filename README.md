# Autoexpress client app

This is the frontend app for the Autoexpress company. This is set up to be deployed along with the autoexpress-server app and interact with the bussiness API.

## Instalation 

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```
